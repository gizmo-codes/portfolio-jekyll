Current Projects

NPI Checker

NPI chekcer allows you to search for any doctor in the U.S. through a government API. Doctor information (API1) and insurance information (API2) is then combined in a table and displayed to the user.

This information is needed and used in a mission critical environment, therefore me and a partner implemented a more reliable SQLite database backup/query that triggers when either API query fails.

NPI Checker is the most complete project I have worked on all the way to this self deployment version. I have several different versions of this project on the GitHub below as I contuned improving the product to learn dockers/deployment.

Live Project | GitHub

SMIDGen

SMIDGen is another web-app used to query the twitter API and return results based on your search. SMIDGen features authentication, user, project, and data management through firebase. You are able to save the results of your query to a database tied to your project.

This app is still very much in-progress, and the full vision includes analyzing the data through machine learning/AI models that suits the users needs.

This was my senior capstone project at UD that is being carried on next semester.
