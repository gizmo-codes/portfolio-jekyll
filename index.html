Current Projects

NPI Checker

NPI checker allows you to search for any doctor in the U.S. through a government API. Doctor information (API1) and insurance information (API2) is then combined in a table and displayed to the user.

This information is needed and used in a mission critical environment, therefore I, along with a partner, implemented a more reliable SQLite database backup/query that triggers when either API query fails.

NPI Checker is the most complete and well rounded project I have worked on. I have several different versions of this project on the GitHub account below as I continue improving the product to learn dockers, deployment and other technologies.

Live Project | GitHub

SMIDGen

SMIDGen is another web-app used to query the twitter API and return results based on your search. SMIDGen features authentication, user, project, and data management through firebase. You are able to save the results of your query to a database tied to your project.

This project is still very much a work in-progress, and the full vision includes analyzing any set of data through machine learning/AI models that suits the users needs.

SMIDGen was my capstone group project at UD that is being carried on next semester. I produced the Twitter API interaction (Python flask API), the search page, and saving/loading data from Firebase.
